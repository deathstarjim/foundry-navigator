Foundry Navigator

Foundry Navigator adds keyboard-first and screen-reader-friendly improvements to Foundry VTT, with the current implementation focused on D&D 5e on Foundry VTT v13.

Current Scope

  • Foundry VTT v13
  • D&D 5e system
  • Tidy 5e Sheets and default D&D 5e actor sheets

What It Does

Character Sheet Keyboard Navigation

  • Alt+T returns focus to the active sheet tab
  • Tab and Shift+Tab move between supported tab controls
  • Enter activates tabs
  • Lets keyboard users move into the active panel and cycle through controls more predictably
  • Escape and Ctrl+Shift+Tab help users leave sheet focus modes cleanly
  • Provides spoken guidance when keyboard focus first enters a supported character sheet

Supported today:

  • Default D&D 5e actor sheets
  • Tidy 5e character sheets

Inventory, Spell, and Action Flows

  • Keyboard navigation for inventory row actions
  • Keyboard support for spell and feature rows
  • Support for attack and use/roll buttons
  • Item context menu support on Tidy 5e sheets
  • Improved focus handling through roll configuration dialogs
  • Restores focus back to the originating row after many attack and consumable flows

Keyboard-First Target Selection and Combat Follow-Up

  • Keyboard target selection dialogs
  • Distance-aware target ordering
  • Self-first consumable targeting when appropriate
  • Hit or miss follow-up dialogs
  • Direct damage and healing application flows
  • GM-mediated damage and healing application for players acting on unowned hostile tokens

Screen Reader Announcements

  • Chat messages and roll results
  • Combat turn changes
  • UI notifications
  • Token movement
  • Tokens entering or leaving the current scene
  • HP and damage changes on owned actors
  • Status effect and condition changes on owned actors
  • Alt+R re-reads the most recent roll result

Helpful Keyboard Shortcuts

  • Alt+T
  • Alt+R
  • Alt+C
  • Alt+Shift+A
  • Alt+Shift+K
  • W
  • Enter and Shift+Enter
  • Many shortcuts can be changed through Foundry's controls configuration

Installation

Install using the module manifest from the latest GitHub release. For development builds, make sure the published module.json and downloadable module zip stay in sync and that module.json is at the root of the zip.

Limitations

  • The strongest support is currently on D&D 5e
  • Tidy 5e and default D&D 5e sheets are the primary supported sheet types
  • Navigation and screen-reader support is still actively evolving

Acknowledgments

This project was inspired in part by earlier Foundry accessibility work from Cora (silvative), including the Accessibility Enhancements module. Foundry Navigator is maintained as its own D&D 5e-focused module.

Bugs and Enhancements

Please submit bug reports and enhancement requests through GitHub Issues.

Helpful details include:

  • Foundry version
  • System version
  • Sheet type in use
  • Other relevant modules
  • Clear reproduction steps

For enhancement requests, it helps to describe the table workflow problem you are trying to solve and what the ideal keyboard or screen reader workflow would look like.
